# AI Chat

A conversational AI web app powered by the Gemini API.

## Tech Stack

-   React + Vite
-   Tailwind CSS
-   Gemini API (Google)
-   Express.js

## Run locally

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your `GEMINI_API_KEY`
4. Run `node server.js` and `npm run dev:client` in two terminals
